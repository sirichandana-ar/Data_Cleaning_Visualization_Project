# Data Cleaning & Sales Visualization Project

An automated Python dashboard that cleans transactional data, exports structured CSV files, and generates visual sales analytics.

## Tech Stack
* **Language:** Python
* **Database:** SQLite3
* **Libraries:** Pandas, Matplotlib, Faker

## Project Structure
* `sales_dashboard.py` - Main execution script
* `sales_dashboard.png` - Exported dashboard visualization
* `*.csv` - Cleaned data exports categorized by region, products, segments, and monthly trends
